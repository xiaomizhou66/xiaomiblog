2.1 原因

原来是跟 Less 的语法冲突了，Less 把 calc 的参数当作运算式执行了。解决方式是使用 Less 的转义字符~。它的具体用法可以看文章结尾的参考阅读中的 Less Language Escaping。文档中说到：GIT:https://github.com/SomMeri/less4j/wiki/Less-Language-Escaping 这个文档就有说明.
