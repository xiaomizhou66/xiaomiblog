react.js--生命周期 与 对应周期函数

Aug 19, 2018⑤ react.jsreact.js 框架学习

一、前言

生命周期中 render() 是必须的，这是最简单的 class 创建的组件。

每一个组件都有几个你可以重写以让代码在处理环节的特定时期运行的 “生命周期方法”。

方法中带有前缀 will 的在特定环节 之前被调用，
而带有前缀 did 的方法则会在特定环节 之后被调用。

二、生命周期

React生命周期主要包括三个阶段：初始化阶段、运行中阶段和 销毁阶段，在React不同的生命周期里，会依次触发不同的钩子函数。
