React 元素的事件处理和 DOM 元素的很相似。但是有一点语法上的不同:

  • React 事件绑定属性的命名采用 驼峰式写法，而不是小写。因为 React 是 JS ，而 DOM 元素 是 html
  • 如果采用 JSX 的语法你需要传入一个函数作为事件处理函数，而不是一个字符串(DOM元素的写法)
